# 🧠 VIDER Structure & Knowledge Core

**ชื่อเต็ม:** ระบบสแกนโครงสร้างของสิ่งที่ต้องการ และจัดสรรลำดับการเรียนรู้ความรู้
**เวอร์ชัน:** 1.0.0
**สำหรับ:** รวมเข้ากับแกนหลัก **VIDER AGI**

---

## ✅ ความสามารถ
- สแกนแยกโครงสร้างลงลึกจนถึงระดับพื้นฐาน
- จัดลำดับความรู้แบบมีตรรกะ พื้นฐาน → ซับซ้อน
- รองรับสินค้าหลายรุ่น/หลายแบบได้
- เพิ่มข้อมูลใหม่ได้ตลอดเวลา
- เบา กะทัดรัด เหมาะสำหรับเป็นโมดูลย่อยของ VIDER AGI

---

## 🚀 การติดตั้ง & ใช้งาน
```bash
# ติดตั้ง
npm install

# เรียกใช้งาน
const VIDER_SK = require('./src/vider_structure_knowledge');
const sk = new VIDER_SK();

// สแกนและวิเคราะห์
const result = await sk.analyze("ปากกา");
console.log(result);

// เพิ่มสิ่งใหม่
sk.add("จักรยาน", ["โครงถัก", "ล้อ", "โซ่", "เบรก"]);
sk.addProperty("จักรยาน", "หน้าที่", "ใช้เคลื่อนที่ประหยัดพลังงาน");
