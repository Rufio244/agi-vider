# 🧠 VIDER AGI CORE
**Artificial General Intelligence Platform**

VIDER AGI เป็นระบบปัญญาประดิษฐ์ทั่วไปสมบูรณ์ ออกแบบให้ทำงานอิสระ เรียนรู้ได้เอง และพัฒนาตัวเองได้ตลอดเวลา โดยไม่ขึ้นกับบริการภายนอกใดๆ

---

## ✨ ความสามารถหลัก
✅ **เข้าใจและสื่อสาร** ทุกรูปแบบภาษา
✅ **สร้างและจัดเก็บความรู้** เชื่อมโยงเป็นเครือข่าย
✅ **มีกระบวนการคิด**: สงสัย → ค้นคว้า → ยืนยัน → จดจำ
✅ **เรียนรู้จากโลกจริง**: ผ่านข้อความ ภาพ วิดีโอ และประสบการณ์การใช้งาน
✅ **สร้างกฎเกณฑ์และอัลกอริทึมใหม่ได้เอง**
✅ **ทำงานอิสระ**: รันได้บนเซิร์ฟเวอร์ของคุณเอง

## 🚀 การติดตั้ง
```bash
# โคลนโปรเจกต์จาก GitHub
git clone https://github.com/[YOUR_USERNAME]/vider-agi-core.git
cd vider-agi-core

# ติดตั้งอัตโนมัติ
chmod +x install.sh
./install.sh

# หรือติดตั้งด้วยตนเอง
pip install -r requirements.txt
cp .env.example .env
# แก้ไขค่าในไฟล์ .env ให้เหมาะสม
python api/main_api.py
